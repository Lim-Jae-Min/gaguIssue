## 프로젝트 선정이유
현재, 학원에 소속되지 않은 강사들은 레슨생을 모집하기 쉽지 않은 상태이다. 악기 레슨을 한다는 것을 입소문으로만 홍보를 하거나 네이버
블로그, 카카오톡으로 홍보하는 것이 대부분이다. SONA 를 통해 자신의 역량을 홍보하고, 학원에 소속된 강사만큼 체계적인 시스템으로 출결 관리를 하여 원활한 강의 활동을 돕고자 한다.
